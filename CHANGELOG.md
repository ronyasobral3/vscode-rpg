ENGLISH VERSION

Release notes formatted for the Visual Studio Code Marketplace.

## [1.4.2] - 2024-08-31

Highlights

- Interactive monster panel with animations and immediate visual feedback.
- Critical hit effects: critical attacks display an animation and larger damage values.
- Dynamic health bar that updates in real time as you type.
- Score counters: track typed characters and defeated monsters.
- Visual and responsive improvements (background, theme variables and minor visual fixes).

Details (notable changes)

- UI: background image, theme variables and style adjustments for better editor integration.
- Visual feedback: damage indicators with animations (float, pulse, shake) and critical text.
- Healthbar: smooth transitions and support for different panel widths.
- Logic: monster state handling and automatic update when life reaches zero.

Compatibility & notes

- Designed to run as a Webview inside VS Code.
- Responsive across different panel sizes; additional adjustments can be requested.

PORTUGUESE VERSION

## [1.4.0] - 2024-08-31

Principais novidades

- Painel interativo do monstro com animações e feedback visual imediato.
- Efeitos de dano críticos: ataques críticos mostram animação e valor maior de dano.
- Barra de vida dinâmica com atualização em tempo real à medida que você digita.
- Contadores de pontuação: acompanha quantas teclas foram digitadas e quantos monstros foram derrotados.
- Melhorias de responsividade e estilo (background, variáveis de tema e pequenas correções visuais).

Detalhes (mudanças relevantes)

- UI: imagem de fundo, variáveis de tema e ajustes de estilo para melhor integração com o editor.
- Feedback visual: indicadores de dano com animações (float, pulse, shake) e textos de crítico.
- Healthbar: indicator com transições suaves e suporte a diferentes larguras de tela.
- Lógica: tratamento de estado do monstro e atualização automática quando a vida chega a zero.

Compatibilidade e notas

- Projetado para funcionar como um Webview dentro do VS Code.
- Responsivo para diferentes tamanhos de painel; ajustes adicionais podem ser solicitados.
